 University Campus Network Design

 Overview

This project presents the design and implementation of a university campus network using Cisco Packet Tracer. The network connects four faculties and a scientific computation center through a hierarchical topology. The project focuses on organizing the network, assigning IP addresses, and providing communication between different departments.

 Network Structure

- 4 Faculties
- Scientific Computation Center
- 4 Floors in each faculty
- 5 Labs on each floor
- 10 PCs in each lab
- Hierarchical Network Topology

Features

- VLAN Configuration
- IP Addressing
- Subnetting
- Inter-VLAN Routing
- Trunk Configuration (802.1Q)
- Connectivity Testing

Technologies Used

- Cisco Packet Tracer
- VLANs
- IP Addressing
- Subnetting
- Routing
- Switching

Testing

The network was tested to verify:

- Communication between devices in the same VLAN
- Communication between different VLANs
- Access to the Scientific Computation Center servers
- Packet delivery using Traceroute
